# Output Structure

## Memory Mode

- **Convert**: `memory/<name>/<rel>/converted/` (markdown + images)
- **Chunk**: `memory/<name>/<rel>/chunked/` (chunked markdown)

## Pipeline Mode (single-folder)

- **Convert**: `pipeline/converted/`
- **Chunk**: `pipeline/chunked/`
