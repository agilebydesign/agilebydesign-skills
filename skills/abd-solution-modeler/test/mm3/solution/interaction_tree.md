# Interaction Tree — Epic Skeleton

## Epics

- **Create Character** — **Character** acquires **Ability**, **Skill**, **Advantage**, **Power** via **PowerPoint**
- **Acquire Power** — **Character** spends **PowerPoint** on **Power** built from **Effect**, **Modifier**, **Descriptor**, **Array**
- **Acquire Skill** — **Character** spends **PowerPoint** on **Skill**
- **Acquire Advantage** — **Character** spends **PowerPoint** on **Advantage**
- **Resolve Check** — **Character** rolls **Check** vs **DifficultyClass**; **Degree** of success/failure
- **Run Action Round** — **Character** acts in **ActionRound**; **Initiative** orders turns
- **Resolve Attack** — **Character** makes **Attack**; **Defense** opposes; **Effect** resisted by **Resistance**; **Condition** results
- **Use Equipment** — **Character** uses **Device**, **Equipment**, **Weapon**, **Vehicle**, **Headquarters**
