MIX AND MATCH PSYCHIC POWERS

The Psychics you can create using these tables tend to
be very specialized, especially the Telekinetic and Tele-
pathic.  If  you’re  interested  in  playing  a  hero  who  has
both telekinetic and telepathic powers, take the Telepa-
thy  power,  then  take  the Telekinesis  power  as  the  first
power in an array and roll for (or choose) three alternate
effects off whichever lists you want.
