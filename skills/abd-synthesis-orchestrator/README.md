# abd-synthesis-orchestrator

Drives the synthesis workflow (create_strategy → run_slice → validate) using abd-story-synthesizer. Works on any engine specified by `conf/abd-config.json`.

## Run

```bash
python C:\dev\agilebydesign-skills\skills\abd-synthesis-orchestrator\scripts\run_orchestrator.py --skill-space <path-to-skill-space>
```

See [SKILL.md](./SKILL.md) for options and [docs/](./docs/) for design docs.
